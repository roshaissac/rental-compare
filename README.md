# Rental Comparison Tool

Single-file, client-side rental comparison tool for renters - a complimentary gift from
Sheena & Rosh Issac Property Group (Associates of Dreamhouse Realty Ltd.), linked from the
rental-viewing follow-up email as a decision aid and a longer-term nurture asset.

- **File:** `index.html` - open directly in a browser, no build step, no server needed.
- **Data:** everything stays in the browser (localStorage, this device only). Nothing is
  saved, sent, or tracked.
- **Accent:** Honey Gold `#c9962e` - confirmed by Rosh as this app's signature colour.
- **Font:** Fraunces self-hosted at `assets/fraunces-latin-var.woff2` (zero third-party on
  first paint - suite guardrail).

## The Issac House Style

Reconciled against the canonical design system (`DESIGN-SYSTEM.md` + `mockups/kit.css` in
`IssacRealty_DigitalCard`): app-shell (stationary dark hero -> scrolling ivory workspace ->
stationary RECA brokerage band), kit tokens/radii/shadows, gold-gradient CTA, warm-white
text on dark, AA-checked accent pairings, `theme-color #09090b`.
